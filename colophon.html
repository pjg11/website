This site is built with Jekyll using a custom theme - inspired by the Seredyna theme - and hosted on GitHub Pages.

The site uses system fonts for the text and code and tries to rely on browser defaults for styling where possible, which help keep the CSS compact.

The color scheme is inspired from minimal text editor themes - quiet for vim and Almost Mono for Emacs.
